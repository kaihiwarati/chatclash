# 🤖 Telegram Ranking Bot
Its dead now but you can use code

A powerful Telegram bot with ranking, leaderboard, and admin system.

---

## 🚀 Features

* 🏆 Daily & Lifetime Ranking System
* 📊 User Stats Tracking
* 🌍 Global & Group Leaderboards
* 👥 Multi-group tracking
* 👮 Admin commands (ban, mute, promote, etc.)
* 👋 Welcome & Leave messages

---

## ⚡ Commands

### 👤 User Commands

* `/start` - Start the bot
* `/ping` - Check bot status
* `/stats` - View your stats
* `/top` - Group leaderboard
* `/global` - Global leaderboard
* `/groups` - Top groups
* `/mygroups` - Your top groups

### 👮 Admin Commands

* `/ban` - Ban user
* `/mute` - Mute user
* `/unmute` - Unmute user
* `/promote` - Promote user
* `/demote` - Demote user

---

## 🔐 Setup

1. Clone repo
2. Create `.env` file
3. Add your bot token:

API_TOKEN=your_token_here

4. Run:

python chat.py
